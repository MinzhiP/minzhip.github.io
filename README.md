# Michelle P Portfolio (GitHub Pages)

This repository is configured as a static GitHub Pages site using plain HTML and CSS.

## Publish with GitHub Pages

Follow GitHub's branch-based Pages instructions:

1. Push this repository to GitHub.
2. In GitHub, open **Settings** for this repository.
3. Go to **Pages** in the sidebar.
4. Under **Build and deployment**:
   - **Source**: Deploy from a branch
   - **Branch**: `main`
   - **Folder**: `/ (root)`
5. Save and wait for deployment.
6. Your site will be available at:
   - `https://<your-github-username>.github.io/minzhip.github.io/`

## Local preview

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
